# esp32-sketches-IoT-thkoeln

* Grove - Ultrasonic Ranger
* Grove - Round FOrce Sensor FSR402
* Grove - LED
* NodeMCU ESP32

## [fourLampPrototype.cpp](https://github.com/floseph/esp32-sketches-IoT-thkoeln/blob/main/src/fourLampPrototype.cpp)
